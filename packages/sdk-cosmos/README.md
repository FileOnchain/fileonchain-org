# @fileonchain/sdk-cosmos

FileOnChain Cosmos anchor client — anchor file and folder CIDs via transaction memos — dependency-free, bring a structural signer.
Exports `anchorChunkedFile` with the uniform progress/receipt shape shared
by every family, built on the [`@fileonchain/utils`](../utils) core.

Part of the FileOnChain SDK — see the [`@fileonchain/sdk`](../sdk) umbrella
(this package is its `@fileonchain/sdk/cosmos` subpath) and the
[repository](https://github.com/FileOnchain/fileonchain-org) for usage docs.
